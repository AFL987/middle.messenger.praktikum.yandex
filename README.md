# Учебный чат

## Установка

Node >=16.13.2

Команды:
 - "start": - включение сервера 
 - "server": - запуск сервера
 - "build": - сборка для прода
 - "restart": - перезагрузка сервера 
 - "kill" - закрытие порта
 - "parsel" - сборка для разработки
